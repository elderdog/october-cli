# october-cli
A simple vue cli practice.

## Project setup
```shell
npm install # or yarn
```

### Bind command
```shell
npm link
```

### Unbind command
```shell
npm unlink
```

### Try out
```shell
oct create <app-name>
```

### For more help
```shell
oct -h # or oct --help
```
